# 🚍 UrbanTransit GTFS Builder MX

> Open-source toolkit for building GTFS feeds from GIS data.

---

## 🌎 Mission

UrbanTransit GTFS Builder MX is an open-source project focused on helping cities, municipalities, universities and consultants create high-quality GTFS feeds from geographic data.

The goal is to simplify the publication of public transport information on platforms such as Google Maps.

---

## ✨ Current Features

- ✅ Read KML files
- ✅ Detect stops
- ✅ Detect routes
- ✅ Export `stops.txt`
